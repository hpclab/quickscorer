QuickScorer
===========

**QuickScorer** was designed by Lucchese, C., Nardini, F. M., Orlando, S., Perego, R., Tonellotto, N., and Venturini, R.,
with the support of Tiscali S.p.A. 

The QuickScorer algorithm is discussed in *Lucchese, C., Nardini, F.
